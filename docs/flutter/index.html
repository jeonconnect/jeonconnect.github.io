Conditional Widget

child 에 컨디셔널 위젯 지정

true ? Text('xxx') : null

children 에 컨디셔널 위젯 추가

if(true) ...[
	Text('1'),
	Text('2'),
]

또는

if(true) ...[
	Text('1'),
	Text('2'),
] else ...[
	Text('3'),
	Text('4'),
]
