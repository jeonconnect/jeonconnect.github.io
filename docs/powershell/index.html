설치

MacOS에서 설치하기

설치

brew install --cask powershell

업그레이드

brew update
brew upgrade powershell --cask

참고

  • https://docs.microsoft.com/ko-kr/powershell/scripting/install/installing-powershell-on-macos?view=powershell-7.2
  • https://zigford.org/setting-powershell-as-default-on-macos.htmls
    • 기본 쉘로 설정

Ubuntu 20.04에서 설치하기

# Update the list of packages
sudo apt-get update
# Install pre-requisite packages.
sudo apt-get install -y wget apt-transport-https software-properties-common
# Download the Microsoft repository GPG keys
wget -q https://packages.microsoft.com/config/ubuntu/20.04/packages-microsoft-prod.deb
# Register the Microsoft repository GPG keys
sudo dpkg -i packages-microsoft-prod.deb
# Update the list of packages after we added packages.microsoft.com
sudo apt-get update
# Install PowerShell
sudo apt-get install -y powershell
# Start PowerShell
pwsh

기본문법사항

자동변수

  • 자동 변수 정보 - PowerShell | Microsoft Docs

배열

생성

$a = 1, 2, 3

빈 배열 생성

$a = @()

해시테이블

생성

$a = @{a=1;b=2}

빈 해시테이블 생성

$a = @{}

문자열

이스케이프

이스케이프를 위해 ` 를 사용한다.

$foo = @"
a
b
c
"@.Replace("`r`n","").Replace('a','')

클로저

스크립트 블럭에서 바깥 스코프의 변수를 참조할 수 있다.

$a = 1, 2, 3
$b = { return "a", "b", $a[0] }
& $b #스크립트 블럭 실행

파일시스템

링크

심볼릭 링크

New-Item -ItemType SymbolicLink -Path "Link" -Target "Target"

하드링크

New-Item -ItemType HardLink -Path "Link" -Target "Target"

참고

  • Create Symbolic Link in Windows 10 with PowerShell (winaero.com)

삭제

삭제시 -Force 스위치를 붙이지 않으면 권한 문제가 발생하는 경우가 많다.

Remove-Item "c:\some\path" -Recurse -Force

참고

  • https://stackoverflow.com/a/25806947

원격

ssh

아래와 같은 방식으로 원격지의 ssh 서버에 대해서 여러 명령을 실행시킬 수 있다.

ssh -tt username@some.server.com @"
pwd
sudo ls /etc
"@.replace("`r","")

명령어

were.exe

실행파일의 위치를 알려준다. where 는 다른 명령어의 alias 로 등록되어 있기 때문에 where.exe 로 실행해야 한다.

# 예시
where.exe flutter dart
