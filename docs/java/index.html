OpenJDK 설치

MacOS

  • brew 를 통해 설치한다.
brew tap AdoptOpenJDK/openjdk
brew install --cask adoptopenjdk8-openj9

  • 설치할 버전을 선택할 수 있다.

    • https://github.com/AdoptOpenJDK/homebrew-openjdk#available-versions

  • Cask adoptopenjdk8 exists in multiple taps 에러가 발생하면 다음과 같이 조치한다.

    rm /usr/local/Homebrew/Library/Taps/homebrew/homebrew-cask-versions/Casks/adoptopenjdk8.rb
    • https://github.com/AdoptOpenJDK/homebrew-openjdk/issues/106

  • 설치되어있는 JDK 리스트 확인

    /usr/libexec/java_home -V

  • shell profile 설정

    export JAVA_HOME=`/usr/libexec/java_home -v 1.8.0_292`
