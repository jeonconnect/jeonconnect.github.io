방화벽

상태 확인

방화벽 활성화 상태를 조회한다.

sudo ufw status

ssh 서버

설치

sudo apt update
sudo apt install openssh-server

방화벽이 활성화되어있다면 ssh 접근을 허용한다.

sudo ufw allow ssh

제어

sudo systemctl status ssh
sudo systemctl enable ssh
sudo systemctl start ssh
